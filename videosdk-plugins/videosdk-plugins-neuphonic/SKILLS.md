# videosdk-plugins-neuphonic

## Purpose
Integrates Neuphonic TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `NeuphonicTTS` | `TTS` | Neuphonic text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | Neuphonic TTS synthesis |

## Environment Variables
- `NEUPHONIC_API_KEY` — Required for Neuphonic services

## Installation
```bash
pip install videosdk-plugins-neuphonic
```
