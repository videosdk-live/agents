# videosdk-plugins-navana

## Purpose
Integrates Navana speech-to-text with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `NavanaSTT` | `STT` | Navana speech-to-text |

## Key Files

| File | Description |
|------|-------------|
| `stt.py` | Navana STT implementation |

## Environment Variables
- `NAVANA_API_KEY` — Required for Navana services

## Installation
```bash
pip install videosdk-plugins-navana
```
