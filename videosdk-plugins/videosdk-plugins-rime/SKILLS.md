# videosdk-plugins-rime

## Purpose
Integrates Rime AI TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `RimeTTS` | `TTS` | Rime AI text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | Rime AI TTS synthesis |

## Environment Variables
- `RIME_API_KEY` — Required for Rime AI services

## Installation
```bash
pip install videosdk-plugins-rime
```
