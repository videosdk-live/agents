# videosdk-plugins-groq

## Purpose
Integrates Groq AI TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `GroqTTS` | `TTS` | Groq text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | Groq TTS synthesis |

## Environment Variables
- `GROQ_API_KEY` — Required for Groq services

## Installation
```bash
pip install videosdk-plugins-groq
```
