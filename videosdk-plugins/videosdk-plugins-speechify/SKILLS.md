# videosdk-plugins-speechify

## Purpose
Integrates Speechify TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `SpeechifyTTS` | `TTS` | Speechify text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | Speechify TTS synthesis |

## Environment Variables
- `SPEECHIFY_API_KEY` — Required for Speechify services

## Installation
```bash
pip install videosdk-plugins-speechify
```
