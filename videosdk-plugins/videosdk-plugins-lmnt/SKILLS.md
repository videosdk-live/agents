# videosdk-plugins-lmnt

## Purpose
Integrates LMNT AI TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `LMNTTTS` | `TTS` | LMNT AI text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | LMNT TTS synthesis |

## Environment Variables
- `LMNT_API_KEY` — Required for LMNT services

## Installation
```bash
pip install videosdk-plugins-lmnt
```
