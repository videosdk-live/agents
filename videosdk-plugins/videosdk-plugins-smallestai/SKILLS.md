# videosdk-plugins-smallestai

## Purpose
Integrates Smallest AI TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `SmallestAITTS` | `TTS` | Smallest AI text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | Smallest AI TTS synthesis |

## Environment Variables
- `SMALLESTAI_API_KEY` — Required for Smallest AI services

## Installation
```bash
pip install videosdk-plugins-smallestai
```
