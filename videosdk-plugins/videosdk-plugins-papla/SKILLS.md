# videosdk-plugins-papla

## Purpose
Integrates Papla Media TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `PaplaTTS` | `TTS` | Papla Media text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | Papla TTS synthesis |

## Environment Variables
- `PAPLA_API_KEY` — Required for Papla services

## Installation
```bash
pip install videosdk-plugins-papla
```
