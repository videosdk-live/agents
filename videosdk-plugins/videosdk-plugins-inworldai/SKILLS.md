# videosdk-plugins-inworldai

## Purpose
Integrates InWorld AI TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `InworldAITTS` | `TTS` | InWorld AI text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | InWorld AI TTS synthesis |

## Environment Variables
- `INWORLD_API_KEY` — Required for InWorld AI services

## Installation
```bash
pip install videosdk-plugins-inworldai
```
