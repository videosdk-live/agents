# videosdk-plugins-humeai

## Purpose
Integrates Hume AI expressive TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `HumeAITTS` | `TTS` | Hume AI expressive text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | Hume AI TTS with expressive voice synthesis |

## Environment Variables
- `HUME_API_KEY` — Required for Hume AI services

## Installation
```bash
pip install videosdk-plugins-humeai
```
