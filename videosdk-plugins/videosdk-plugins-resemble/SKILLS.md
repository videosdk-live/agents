# videosdk-plugins-resemble

## Purpose
Integrates Resemble AI TTS with the VideoSDK AI Agents framework.

## Exported Classes

| Class | Base Class | Description |
|-------|-----------|-------------|
| `ResembleTTS` | `TTS` | Resemble AI text-to-speech |

## Key Files

| File | Description |
|------|-------------|
| `tts.py` | Resemble AI TTS synthesis |

## Environment Variables
- `RESEMBLE_API_KEY` — Required for Resemble AI services

## Installation
```bash
pip install videosdk-plugins-resemble
```
